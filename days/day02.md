
# Day 2 – LangChain Primer: Models, Prompts, Chains

_Timebox: ~1 hour_

## Knowledge goals

- Understand LangChain’s core concepts: models, prompts, chains
- Learn how to structure a basic chain and prompt
- Run your first LLM call using LangChain

## Learning materials (≈ 20–25 minutes)

- Read: [**LangChain — Overview**](https://docs.langchain.com/oss/python/langchain/overview)
- Read the definition and examples of each components

## Activities (about 30 minutes)

- Write a one-paragraph summary of LangChain’s core concepts (models, prompts, chains)
- Sketch a diagram showing how a prompt flows through a model and chain
- Capture notes in `notes/day02_notes.md` (or your journal) for future reference.

## Knowledge check

- In your words: what is a LangChain chain and why use it?
- Describe the flow from prompt to model to tool output
- Definition of the core components
